# Day 3 - Question Bank Engine

## Objective

Build the core question management system for ExamBooster.

## Completed Features

* Question Model Creation
* Add Question API
* Get Questions API
* Subject Filtering
* Difficulty Filtering
* Random Question Generator

## APIs Implemented

POST /api/questions/add

GET /api/questions

GET /api/questions?subject=...

GET /api/questions?difficulty=...

GET /api/questions/random?limit=...

## Outcome

ExamBooster can now store, retrieve, filter, and randomly generate questions from MongoDB.
