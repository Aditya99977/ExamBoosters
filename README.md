# 🚀 ExamBooster

A modern **MERN Stack** platform for government exam preparation, featuring mock tests, performance analytics, and personalized learning.

> **Status:** 🚧 Active Development

---

## ✨ Features

- 🔐 JWT Authentication & Role-Based Access
- 👤 User Profile Management
- 📊 Student Dashboard
- 📝 Mock Test System
- 📈 Performance Analytics
- 📚 Question Bank
- 👨‍💼 Admin Dashboard
- 📱 Responsive UI

---

## 💻 Tech Stack

### Frontend
- React
- Vite
- Bootstrap 5
- Chart.js
- Axios
- React Router

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- bcryptjs

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Aditya99977/ExamBoosters.git
cd ExamBoosters
```

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
