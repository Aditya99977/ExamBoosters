# 🚀 ExamBooster

A modern **MERN Stack** platform for government exam preparation that helps students practice questions, attempt mock tests, analyze performance, and prepare smarter using AI-powered learning.

> **Status:** 🚧 Beta Development

---

# ✨ Features

## 👨‍🎓 Student Features

- 🔐 JWT Authentication
- 👤 User Profile
- 🏠 Student Dashboard
- 📚 Practice Questions
- 📝 Mock Test System
- 📄 Previous Year Papers
- 📈 Performance Dashboard
- 🤖 AI Tutor
- 📱 Fully Responsive UI

---

## 👨‍💼 Admin Features

- Admin Dashboard
- Question Management
- Previous Year Papers Management
- CSV Bulk Upload
- User Management
- Paper Management
- Dashboard Analytics

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/Aditya99977/ExamBoosters.git

cd ExamBoosters
```

---

## Backend

```bash
cd backend

npm install

npm run dev
```

Backend runs on:

```
http://localhost:5000
```

---

## Frontend

```bash
cd frontend

npm install

npm run dev
```

Frontend runs on:

```
http://localhost:5173
```
